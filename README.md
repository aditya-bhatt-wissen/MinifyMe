# 🚀 MinifyMe — URL Shortener Service

MinifyMe is a lightweight, fast, and scalable **URL shortening service** built with **Spring Boot**. It converts long URLs into short, shareable links while tracking analytics and supporting optional expiry times.

---

## ✨ Features

- Shorten any long URL to a compact link
- Redirect to the original URL instantly
- Track click counts for each short URL
- Set optional expiry times for links
- Simple, clean REST API for easy integration

---

## 🛠️ Tech Stack

- **Java 17+**  
- **Spring Boot 3+**  
- **Spring Web & Spring Data JPA**  
- **Hibernate**  
- **MySQL / H2 Database**  
- **Lombok**  
- **Maven**
